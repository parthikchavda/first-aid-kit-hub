નેટવર્ક ઉપકરણો — Network Devices
Hardware devices used in computer networks
LAN

LAN — Local Area Network

Local Area Network

નાના વિસ્તારમાં (ઘર, ઓફિસ, શાળા) computers ને જોડતું નેટવર્ક. ઝડપ ઊંચી, ખર્ચ ઓછો.

ઉદાહરણ: ઘરનું Wi-Fi, ઓફિસ નેટવર્ક | Speed: 100Mbps–10Gbps
WAN

WAN — Wide Area Network

Wide Area Network

મોટા ભૌગોલિક વિસ્તારને આવરી લેતું નેટવર્ક. ઇન્ટરનેટ એ WAN નું સૌથી મોટું ઉદાહરણ છે.

ઉદાહરણ: Internet, BSNL/Jio નેટવર્ક | ઘણા શહેરો/દેશો
MAN

MAN — Metropolitan Area Network

Metropolitan Area Network

એક શહેર અથવા કેમ્પસ સ્તરે ઉપયોગ. LAN અને WAN ની વચ્ચેનો આ network type છે.

ઉદાહરણ: City-wide cable TV network, University campus | ~50km
WLAN

WLAN — Wireless LAN

Wireless Local Area Network

Wi-Fi technology દ્વારા wire વિના LAN connection. IEEE 802.11 standard ઉપયોગ.

ઉદાહરણ: Home Wi-Fi router, cafe Wi-Fi hotspot
PAN

PAN — Personal Area Network

Personal Area Network

એક વ્યક્તિ આસપાસ (~10 meter) ના devices ને જોડે. Bluetooth / USB ઉપયોગ.

ઉદાહรণ: Phone–Earphone Bluetooth, Smartwatch connection
VPN

VPN — Virtual Private Network

Virtual Private Network

Encrypted tunnel through internet. Private, secure communication over public network.

ઉદાહरण: Work-from-home office access, NordVPN, ExpressVPN

📦 Network Hardware Devices

Router

Router — રાઉટર

Networks ને connect કરે અને packets ને best path via IP address routing table ઉપર મોકલે.

ઉદાહርણ: Jio Fiber router ઘરમાં | Layer 3 device
Switch

Switch — સ્વિચ

LAN ની અંદર MAC address ઉપર based data frames forward કરે. Smart, intelligent hub.

ઉદાહरण: 24-port office switch | Layer 2 device
Hub

Hub — હબ

Data બધા ports ને broadcast કરે, destination ને નહીં. Dumb device, ઓછો ઉપયોગ.

Layer 1 device | Half-duplex | Legacy technology
Bridge

Bridge — બ્રિજ

બે LAN segments ને MAC address ઉપર based join કરે. Collision domains ઘટાડે.

Layer 2 device | Wireless bridge | Network segment join
Gateway

Gateway — ગેટવે

અલગ protocols અથવા networks ને translate/connect કરે. Internet access point.

ઉદાહรણ: Home router = default gateway (192.168.1.1)
Repeater

Repeater — રીપીટર

Signal weak થાય ત્યારે amplify/regenerate કરે. Long cable distances માટે.

Layer 1 device | Wi-Fi range extender | Optical repeater
NIC

NIC — Network Interface Card

Network Interface Card

Computer ને network સાથે physically connect કરે. Unique MAC address ધરાવે.

ઉદાહरण: Ethernet port, Wi-Fi card | MAC: 00:1A:2B:3C:4D:5E
Modem

Modem — મોડેમ

Modulator-Demodulator

Digital signal ↔ Analog signal conversion. ISP connection માટે ઉપયોગ.

ઉદાহरण: DSL modem, Cable modem, 4G USB dongle
AP

Access Point — એક્સેસ પોઈન્ટ

Wireless Access Point (WAP)

Wired network ને wireless clients સુધી extend કરે. Wi-Fi signal provide.

ઉદાહरण: Mall/Airport Wi-Fi APs | Enterprise campus Wi-Fi
Firewall

Firewall — ફાયરવૉલ

Network traffic filter કરે, unauthorized access block કરે. Hardware/Software બંને.

ઉદाहरण: Cisco ASA, Windows Firewall, pfSense
DNS Server

DNS Server

Domain Name System

Domain name → IP address translate. Internet ની "Phone Book".

google.com → 142.250.80.46 | 8.8.8.8 (Google DNS)
DHCP Server

DHCP Server

Dynamic Host Configuration Protocol

Automatic IP address, subnet, gateway, DNS assign કરે devices ને.

Router automatically 192.168.1.X IP આપે | Lease time

IP Addressing — આઈ.પી. સરનામું
IPv4 and IPv6 — Internet Protocol Versions
IPv4

IPv4 — Internet Protocol version 4

32-bit address | ~4.3 billion addresses

4 octets (0–255) ના dotted decimal format. નેટવર્ક ઓળખ માટે.

Format: 192.168.1.1 | Network.Host | Subnet mask: 255.255.255.0
IPv6

IPv6 — Internet Protocol version 6

128-bit address | 340 undecillion addresses

IPv4 exhaustion solution. 8 groups of 4 hex digits. IoT, future internet.

Format: 2001:0db8:85a3:0000:0000:8a2e:0370:7334

Network Protocols — નેટવર્ક પ્રોટોકોલ
Rules for communication between computers

🔵 Transport Layer Protocols

TCP

TCP — Transmission Control Protocol

Connection-oriented. Reliable, ordered delivery. 3-way handshake (SYN→SYN-ACK→ACK). Error checking & retransmission.

ઉdaHaraN: Web browsing, Email, File download | Port varies
UDP

UDP — User Datagram Protocol

Connectionless. Fast, no reliability guarantee. No handshake, no retransmission. Low latency.

ઉdaHaraN: Video call, Gaming, DNS, Streaming | Speed priority

🌐 Application Layer Protocols

HTTP/HTTPS

HTTP / HTTPS

HyperText Transfer Protocol (Secure)

Web pages browse. HTTP = plain text (Port 80). HTTPS = SSL/TLS encrypted (Port 443).

ઉdaHaraN: google.com browse, online shopping secure payment
FTP/SFTP

FTP / SFTP

File Transfer Protocol / Secure FTP

Files upload/download. FTP = Port 20,21. SFTP = Port 22 (SSH based, encrypted).

ઉdaHaraN: Website files upload to server, FileZilla
DNS

DNS — Domain Name System

Name → IP resolution. UDP Port 53. Recursive & Authoritative servers. Cache.

google.com → 142.250.x.x | 8.8.8.8 (Google), 1.1.1.1 (Cloudflare)
DHCP

DHCP

Dynamic Host Configuration Protocol

Auto IP assign. UDP Port 67 (server), 68 (client). DORA: Discover→Offer→Request→Ack.

ઉdaHaraN: Phone Wi-Fi connect → automatic IP મળ્યું
SSH

SSH — Secure Shell

Encrypted remote terminal access. Replaces Telnet. Port 22. Key-based authentication.

ઉdaHaraN: Linux server remotely access | ssh user@192.168.1.1
Telnet

Telnet

Remote terminal access — unencrypted (insecure). Port 23. Legacy, replaced by SSH.

Network device configure (old) | Security risk — avoid
SNMP

SNMP

Simple Network Management Protocol

Network devices monitor/manage. UDP Port 161. MIB (Management Info Base).

ઉdaHaraN: Router, Switch CPU/bandwidth monitor | Nagios, Zabbix
ICMP

ICMP — Internet Control Message Protocol

Error reporting & diagnostics. ping command uses ICMP Echo Request/Reply.

ping google.com, traceroute/tracert command | No port (Layer 3)
ARP

ARP — Address Resolution Protocol

IP address → MAC address mapping. LAN ની અંદર. ARP table/cache maintain.

arp -a command | 192.168.1.1 → 00:1A:2B:3C:4D:5E
NTP

NTP — Network Time Protocol

Computers ના clocks synchronize. UDP Port 123. Stratum levels (0=atomic clock).

ઉdaHaraN: Server time sync, security certificate validation

🔀 Routing Protocols

RIP

RIP — Routing Information Protocol

Distance-vector routing. Max 15 hops. UDP Port 520. Simple but slow convergence.

RIPv1 (classful), RIPv2 (classless) | Small networks only
OSPF

OSPF — Open Shortest Path First

Link-state routing. Dijkstra algorithm. No hop limit. Fast convergence. Areas.

Enterprise/ISP networks | Protocol 89 | Widely used
BGP

BGP — Border Gateway Protocol

Internet ની "backbone" routing protocol. AS (Autonomous System) between. TCP Port 179.

ISPs between routing | Internet ના 80,000+ routes manage
EIGRP

EIGRP — Enhanced IGRP

Enhanced Interior Gateway Routing Protocol

Cisco proprietary. Hybrid (distance-vector + link-state). DUAL algorithm. Fast.

Cisco networks | Protocol 88 | Bandwidth+delay metric

Mobile Network Generations — 1G થી 5G
Evolution of wireless mobile technology
1980s

1G — First Generation

Speed: 2.4 kbps | Analog technology

ફક્ત voice call. Analog signals. AMPS (Advanced Mobile Phone System). No data. Security ઘણી ઓછી. Roaming problems.

1983 Motorola DynaTAC 8000X — first commercial phone
1991

2G — Second Generation

Speed: 9.6 kbps–144 kbps | GSM/CDMA Digital

Digital voice + SMS + basic data. GSM standard. GPRS (2.5G): 56kbps. EDGE (2.75G): 384kbps. Encryption added.

ઉdaHaraN: SMS texting, basic internet, caller ID | India: Airtel, Jio
2000s

3G — Third Generation

Speed: 384 kbps – 21 Mbps | UMTS/HSPA

Broadband mobile internet. Video calls, apps, mobile browsing. WCDMA technology. HSPA+: up to 42Mbps. Always-on internet.

ઉdaHaraN: YouTube mobile (early), WhatsApp, Google Maps | 2008 era
2010s

4G LTE — Fourth Generation

Speed: 100 Mbps – 1 Gbps | LTE/LTE-A

HD video, VoLTE (Voice over LTE), fast app downloads. OFDMA technology. Low latency ~50ms. India: 2012+ launch, Jio 2016 revolution.

ઉdaHaraN: Netflix HD stream, video call, online gaming | 4G LTE-A: 1Gbps
2020s

5G — Fifth Generation

Speed: 1 Gbps – 20 Gbps | mmWave/Sub-6GHz

Ultra-low latency (1ms), massive IoT, autonomous vehicles, smart cities. Network slicing. Beamforming. NSA/SA architecture. India: 2022 launch.

ઉdaHaraN: AR/VR, self-driving cars, remote surgery, smart factory | Jio 5G, Airtel 5G

6G: Expected 2030. Speed: 1 Tbps. AI-native network. Terahertz frequencies. Holographic communication. Sub-1ms latency.

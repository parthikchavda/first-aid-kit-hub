🎯 KEY MNEMONICS (યાદ રાખવાની ટૂંકી ટ્રિક્સ)

1. OSI 7 લેયર્સ યાદ રાખવાની Trick

"A P S T N D P" (ઉપરથી નીચે)

"All People Seem To Need Data Processing"
A Application (7) · P Presentation (6) · S Session (5) · T Transport (4) · N Network (3) · D Data Link (2) · P Physical (1)

2. TCP/IP 4 લેયર્સ યાદ રાખવાની Trick

"A T I N" → "All The Internet Networks"

A Application · T Transport · I Internet · N Network Access

3. પોર્ટ નંબર્સ યાદ રાખવાની Trick

"HTTP = 80" → H = 8 → 80  |  "HTTPS = 443" → સુરક્ષિત = 443
"FTP = 21" → File Transfer = 21  |  "SMTP = 25" → Send Mail = 25
"DNS = 53" → Domain Name = 53

"H F S D" યાદ રાખો: H HTTP(80) · F FTP(21) · S SMTP(25) · D DNS(53)

4. IP Classes યાદ રાખવાની Trick

"A B C D E" → "A Big Cat Danced Everywhere"

A 1-126 (Large) · B 128-191 (Medium) · C 192-223 (Small) · D 224-239 (Multicast) · E 240-255 (Experimental)

5. Private IP Ranges યાદ રાખવાની Trick

"10. 172.16 192.168"

10.0.0.0 – 10.255.255.255 (એક મોટું)
172.16.0.0 – 172.31.255.255 (સોળથી એકત્રીસ)
192.168.0.0 – 192.168.255.255 (એકસો અઠ્ઠાસી)

6. Network Devices OSI Layers યાદ રાખવાની Trick

"R H B S R G" → "Raju Has Been Selling Real Gold"

R Repeater (L1) · H Hub (L1) · B Bridge (L2) · S Switch (L2) · R Router (L3) · G Gateway (All)

7. TCP vs UDP યાદ રાખવાની Trick

"TCP = Trustworthy" (વિશ્વસનીય)  |  "UDP = Ultra Fast" (ઝડપી)

TCP = Connection-Oriented  |  UDP = Connectionless

8. LAN Technologies યાદ રાખવાની Trick

"E T F" → "Every Teacher Feeds Students"

E Ethernet (10 Mbps) · T Token Ring (16 Mbps) · F FDDI (100 Mbps)

9. Network Security યાદ રાખવાની Trick

"F V E"

F Firewall = Filter (ટ્રાફિક ફિલ્ટર)
V VPN = Vault (ડેટા સુરક્ષિત)
E Encryption = Encode (ડેટા એન્કોડ)

10. High Speed Technologies યાદ રાખવાની Trick

"A F G" → "All Friends Gather"

A ATM (53-byte cells) · F Frame Relay (WAN) · G Gigabit Ethernet (1000 Mbps)

📝 ONE-LINER FACTS

1. OSI = Open Systems Interconnection
2. TCP/IP = Transmission Control Protocol/Internet Protocol
3. IP = Internet Protocol (Layer 3)
4. TCP = Transport Control Protocol (Layer 4)
5. UDP = User Datagram Protocol (Layer 4)
6. HTTP = HyperText Transfer Protocol (Port 80)
7. HTTPS = HTTP Secure (Port 443)
8. FTP = File Transfer Protocol (Port 21)
9. SMTP = Simple Mail Transfer Protocol (Port 25)
10. DNS = Domain Name System (Port 53)
11. ATM = Asynchronous Transfer Mode (53-byte cells)
12. FDDI = Fiber Distributed Data Interface (100 Mbps)
13. CSMA/CD = Carrier Sense Multiple Access with Collision Detection
14. MAC = Media Access Control
15. CIDR = Classless Inter-Domain Routing
16. VPN = Virtual Private Network
17. TTL = Time To Live
18. IHL = Internet Header Length
19. ToS = Type of Service
20. LAN = Local Area Network
21. WAN = Wide Area Network
22. NIC = Network Interface Card
23. DHCP = Dynamic Host Configuration Protocol
24. SSL = Secure Sockets Layer
25. TLS = Transport Layer Security

🎯 PSI EXAM - FINAL REVISION CHECKLIST

Day Before Exam – Quick Revision Points

✅ OSI Model (7 Layers)

  • Physical (Layer 1) – Bits
  • Data Link (Layer 2) – Frames, MAC
  • Network (Layer 3) – Packets, IP
  • Transport (Layer 4) – Segments, TCP/UDP
  • Session (Layer 5) – Connection management
  • Presentation (Layer 6) – Encryption, Compression
  • Application (Layer 7) – HTTP, FTP, SMTP, DNS

✅ TCP/IP Model (4 Layers)

  • Network Access Layer
  • Internet Layer (IP)
  • Transport Layer (TCP/UDP)
  • Application Layer (HTTP, FTP, SMTP, DNS)

✅ Network Devices

  • Repeater – Layer 1
  • Hub – Layer 1
  • Bridge – Layer 2
  • Switch – Layer 2
  • Router – Layer 3
  • Gateway – All Layers

✅ Port Numbers

  • HTTP – 80
  • HTTPS – 443
  • FTP – 21
  • SMTP – 25
  • DNS – 53

✅ IP Classes

  • Class A – 1-126, 255.0.0.0
  • Class B – 128-191, 255.255.0.0
  • Class C – 192-223, 255.255.255.0
  • Class D – 224-239, Multicast
  • Class E – 240-255, Experimental

✅ Private IP Ranges

  • 10.0.0.0 – 10.255.255.255
  • 172.16.0.0 – 172.31.255.255
  • 192.168.0.0 – 192.168.255.255

✅ TCP vs UDP

  • TCP = Reliable, Connection-Oriented
  • UDP = Fast, Connectionless

✅ Firewall vs VPN

  • Firewall = Traffic Filter
  • VPN = Encrypted Tunnel

✅ LAN Technologies

  • Ethernet – 10 Mbps, CSMA/CD
  • Token Ring – 16 Mbps, Token Passing
  • FDDI – 100 Mbps, Token Passing, Fiber

✅ High Speed Technologies

  • ATM – 53-byte cells
  • Frame Relay – WAN
  • Gigabit Ethernet – 1000 Mbps

✅ IPv4 Header

  • Version = 4
  • IHL = 5-15 (20-60 bytes)
  • TTL = 64 default
  • Protocol = TCP(6), UDP(17)

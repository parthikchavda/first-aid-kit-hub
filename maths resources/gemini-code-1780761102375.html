૧. સંખ્યા પદ્ધતિ (Number System)

મૂળભૂત નિયમો

ભાજ્ય (Dividend) ભાજક × ભાગફળ + શેષ
પ્રથમ n પ્રાકૃતિક સંખ્યાનો સરવાળો n(n+1) / 2
પ્રથમ n એકી સંખ્યાનો સરવાળો n²
પ્રથમ n બેકી સંખ્યાનો સરવાળો n(n+1)

લસાઅ અને ગુસાઅ (LCM & HCF)

બે સંખ્યાઓનો ગુણાકાર લસાઅ × ગુસાઅ
અપૂર્ણાંકનો ગુસાઅ અંશનો ગુસાઅ / છેદનો લસાઅ
અપૂર્ણાંકનો લસાઅ અંશનો લસાઅ / છેદનો ગુસાઅ

૨. બીજગણિત (Algebra)

નિત્યસમ (Algebraic Identities)

(a + b)² a² + 2ab + b²
(a - b)² a² - 2ab + b²
a² - b² (a - b)(a + b)
(a + b)³ a³ + b³ + 3ab(a+b)

ઘાત અને ઘાતાંકના નિયમો

aᵐ × aⁿ a^(m+n)
aᵐ ÷ aⁿ a^(m-n)
(aᵐ)ⁿ a^(m×n)
a⁰ 1

૩. ત્રિકોણમિતિ (Trigonometry)

મૂળભૂત ગુણોત્તરો (કાટકોણ ત્રિકોણ માટે)

sin θ સામેની બાજુ / કર્ણ
cos θ પાસેની બાજુ / કર્ણ
tan θ સામેની બાજુ / પાસેની બાજુ
cot θ 1 / tan θ

ત્રિકોણમિતિય નિત્યસમ

sin²θ + cos²θ 1
1 + tan²θ sec²θ
1 + cot²θ cosec²θ

૪. પરિમિતિ અને ક્ષેત્રફળ (2D Geometry)

ચોરસ અને લંબચોરસ

ચોરસનું ક્ષેત્રફળ l²
ચોરસની પરિમિતિ 4l
લંબચોરસનું ક્ષેત્રફળ l × b
લંબચોરસની પરિમિતિ 2(l + b)

ત્રિકોણ અને વર્તુળ

ત્રિકોણનું ક્ષેત્રફળ ½ × પાયો × વેધ
વર્તુળનું ક્ષેત્રફળ πr²
વર્તુળનો પરિધ 2πr

૫. પૃષ્ઠફળ અને ઘનફળ (3D Geometry)

સમઘન અને લંબઘન

સમઘનનું ઘનફળ l³
સમઘનનું કુલ પૃષ્ઠફળ 6l²
લંબઘનનું ઘનફળ l × b × h
લંબઘનનું કુલ પૃષ્ઠફળ 2(lb + bh + hl)

નળાકાર અને શંકુ

નળાકારનું ઘનફળ πr²h
નળાકારની વક્રસપાટી 2πrh
શંકુનું ઘનફળ ⅓ πr²h
શંકુની વક્રસપાટી πrl

ગોલક (Sphere)

ગોલકનું ઘનફળ ⁴/₃ πr³
ગોલકનું પૃષ્ઠફળ 4πr²
અર્ધગોલકનું ઘનફળ ²/₃ πr³

ભાગ ૧: Quantitative Aptitude (૩૦ ગુણ)

૧. સંખ્યા પદ્ધતિ (Number System)

મૂળભૂત સૂત્રો નિયમો
ભાગાકાર નિયમ: ભાજ્ય = (ભાજક × ભાગફળ) + શેષ
પ્રથમ n પ્રાકૃતિક સંખ્યા સરવાળો: n(n+1) / 2
પ્રથમ n એકી સંખ્યા સરવાળો: n²
પ્રથમ n બેકી સંખ્યા સરવાળો: n(n+1)
પ્રથમ n સંખ્યાઓના વર્ગોનો સરવાળો: [n(n+1)(2n+1)] / 6
વિભાજ્યતાની ચાવીઓ (Shortcuts)

૩ ની ચાવી: અંકોનો સરવાળો ૩ થી ભાગી શકાય.
૪ ની ચાવી: છેલ્લા બે અંકો ૪ થી ભાગી શકાય.
૧૧ ની ચાવી: એકી સ્થાનોનો સરવાળો - બેકી સ્થાનોનો સરવાળો = 0 અથવા ૧૧ નો ગુણક.

૨. લ.સા.અ. અને ગુ.સા.અ. (LCM & HCF)

સંબંધ અને અપૂર્ણાંક
મુખ્ય નિયમ: સંખ્યા A × સંખ્યા B = LCM × HCF
અપૂર્ણાંકનો ગુ.સા.અ: અંશનો ગુ.સા.અ / છેદનો લ.સા.અ
અપૂર્ણાંકનો લ.સા.અ: અંશનો લ.સા.અ / છેદનો ગુ.સા.અ

૩. ટકાવારી અને ભાગીદારી (Percentage & Partnership)

ટકાવારી (Percentage) શોર્ટકટ્સ
A ની આવક B કરતા x% વધુ હોય તો B ની ઓછી આવક: [x / (100+x)] × 100%
ક્રમિક ટકાવારી ફેરફાર (Successive %): x + y + (xy / 100)
ભાગીદારી (Partnership)
નફાની વહેંચણી ગુણોત્તર: નફો = રોકાણ × સમય
P₁ : P₂ = I₁T₁ : I₂T₂

૪. નફો-ખોટ (Profit & Loss)

મૂળભૂત અને એડવાન્સ સૂત્રો
નફો % = (નફો / મૂળ કિંમત) × 100
ખોટ % = (ખોટ / મૂળ કિંમત) × 100
વેચાણ કિંમત (નફા વખતે): મૂ.કિં × [(100 + નફો%) / 100]
ચોખ્ખો વળતર (Discount): છાપેલી કિંમત - વેચાણ કિંમત

૫. સાદુ વ્યાજ અને ચક્રવૃદ્ધિ વ્યાજ (SI & CI)

સાદુ વ્યાજ (Simple Interest)
વ્યાજ (I) = (P × R × N) / 100
રાશ (Amount) = P + I
ચક્રવૃદ્ધિ વ્યાજ (Compound Interest)
વ્યાજમુદલ (A) = P × (1 + R/100)^N
૨ વર્ષ માટે CI અને SI નો તફાવત: D = P × (R / 100)²
૩ વર્ષ માટે CI અને SI નો તફાવત: D = P(R/100)² × (3 + R/100)

૬. ગુણોત્તર અને પ્રમાણ (Ratio & Proportion)

ગુણોત્તરના પ્રકારો
મધ્યમ પ્રમાણપદ (Mean Proportion) of a, b: √(a × b)
તૃતીય પ્રમાણપદ (Third Proportion): b² / a
ચતુર્થ પ્રમાણપદ (Fourth Proportion) of a,b,c: (b × c) / a

૭. સમય અને કાર્ય, વેતન અને સાંકળનો નિયમ (Time & Work)

સમય અને કાર્ય શોર્ટકટ
A ને x દિવસ, B ને y દિવસ લાગે, સાથે: (x × y) / (x + y)
કાર્યક્ષમતા (Efficiency): ૧ / સમય
સાંકળનો નિયમ (Chain Rule)
મહત્વપૂર્ણ માસ્ટર સૂત્ર: (M₁ × D₁ × H₁) / W₁ = (M₂ × D₂ × H₂) / W₂

M=માણસો, D=દિવસો, H=કલાક, W=કાર્ય/વેતન

૮. સમય, ઝડપ અને અંતર (Time, Speed & Distance)

મૂળભૂત અને ટ્રેન સબંધી
ઝડપ (Speed) = અંતર / સમય
કિમી/કલાક થી મીટર/સેકન્ડ ફેરવવા: ગુણ્યા ૫/૧૮
મીટર/સેકન્ડ થી કિમી/કલાક ફેરવવા: ગુણ્યા ૧૮/૫
સરેરાશ ઝડપ (સમાન અંતર માટે): 2xy / (x + y)
નદી અને હોડી (Boat & Stream)
પ્રવાહની દિશામાં ઝડપ (Downstream): u + v
પ્રવાહની વિરુદ્ધ દિશામાં (Upstream): u - v

(u = હોડીની શાંત પાણીમાં ઝડપ, v = પ્રવાહની ઝડપ)

૯. મધ્યક, મધ્યસ્થ અને બહુલક (Mean, Median, Mode)

આંકડાશાસ્ત્રના સૂત્રો
મધ્યક (Mean - X̄) = અવલોકનોનો સરવાળો / કુલ સંખ્યા
ત્રણેય વચ્ચેનો સંબંધ (સૌથી Imp): Z = 3M - 2X̄

Z = બહુલક, M = મધ્યસ્થ, X̄ = મધ્યક

૧૦. કૌંસ અને વિસ્તરણ (Brackets & Expansions)

સાદુરૂપ (Simplification) નિયમ
ક્રમનો નિયમ: VBODMAS

V = Vinculum (રેખા કૌંસ)
B = Brackets (કૌંસ: (), {}, [])
O = Of (ના/ની/નું ગુણાકાર)
D = Division (ભાગાકાર)
M = Multiplication (ગુણાકાર)
A = Addition (સરવાળો)
S = Subtraction (બાદબાકી)

૧૧. વર્ગ-મૂળ, ઘન-મૂળ અને ઘાતાંક

ઘાતાંકના નિયમો
aᵐ × aⁿ = a^(m+n)
aᵐ ÷ aⁿ = a^(m-n)
(aᵐ)ⁿ = a^(m×n)
a⁻ᵐ = 1 / aᵐ
a⁰ = 1

૧૨. બહુપદીઓ અને અવયવીકરણ (Polynomials & Identities)

મહત્વના નિત્યસમ
(a + b)² = a² + 2ab + b²
(a - b)² = a² - 2ab + b²
a² - b² = (a - b)(a + b)
(a + b + c)² = a²+b²+c²+2ab+2bc+2ca
a³ + b³ = (a+b)(a²-ab+b²)

૧૩. સુરેખ અને દ્વિઘાત સમીકરણ (Equations)

દ્વિઘાત સમીકરણ: ax² + bx + c = 0
વિવેચક (Discriminant) D = b² - 4ac
બીજ (Roots) શોધવાનું સૂત્ર: x = (-b ± √D) / 2a
બીજનો સરવાળો (α + β) = -b / a
બીજનો ગુણાકાર (α × β) = c / a

૧૪. ક્ષેત્રફળ, પૃષ્ઠફળ અને ઘનફળ (Mensuration Complete)

2D આકારો (ક્ષેત્રફળ અને પરિમિતિ)
ચોરસ ક્ષેત્રફળ / પરિમિતિ: l² / 4l
લંબચોરસ ક્ષેત્રફળ / પરિમિતિ: (l×b) / 2(l+b)
વર્તુળનું ક્ષેત્રફળ / પરિધ: πr² / 2πr
ત્રિકોણનું ક્ષેત્રફળ: ½ × પાયો × વેધ
સમબાજુ ત્રિકોણ ક્ષેત્રફળ: (√3/4) × l²
3D આકારો (પૃષ્ઠફળ અને ઘનફળ)
સમઘન ઘનફળ / પૃષ્ઠફળ: l³ / 6l²
લંબઘન ઘનફળ / પૃષ્ઠફળ: lbh / 2(lb+bh+hl)
નળાકાર ઘનફળ / વક્રસપાટી: πr²h / 2πrh
શંકુનું ઘનફળ / વક્રસપાટી: ⅓ πr²h / πrl
ગોલકનું ઘનફળ / પૃષ્ઠફળ: ⁴/₃ πr³ / 4πr²
અર્ધગોલક ઘનફળ / કુલ પૃષ્ઠફળ: ²/₃ πr³ / 3πr²

૧૫. યામભૂમિતિ અને ત્રિકોણમિતિ (Coordinate & Trigonometry)

યામભૂમિતિ (Coordinate Geometry)
બે બિંદુઓ વચ્ચેનું અંતર: √[(x₂-x₁)² + (y₂-y₁)²]
મધ્યબિંદુના યામ: ((x₁+x₂)/2 , (y₁+y₂)/2)
ત્રિકોણમિતિ નિત્યસમ
sin²θ + cos²θ = 1
1 + tan²θ = sec²θ
1 + cot²θ = cosec²θ

ભાગ ૨: Reasoning Ability (૬૦ ગુણ)

૧. કોડિંગ-ડીકોડિંગ (Coding-Decoding)

A to Z સ્થાન કોડ ટ્રીક

યાદ રાખો શબ્દ: E-J-O-T-Y (સ્થાન અનુક્રમે: 5, 10, 15, 20, 25)
વિરુદ્ધ અક્ષરો શોધવા (Reverse Letters): કોઈપણ અક્ષર અને તેની સામેના વિરુદ્ધ અક્ષરના ક્રમનો સરવાળો હંમેશા 27 થાય. (જેમ કે A=1, Z=26 -> 1+26=27)

૨. લોહીનો સંબંધ (Blood Relation)

ફેમિલી ટ્રી સંજ્ઞાઓ (Family Tree Shortcuts)

🔵 ચોરસ / પ્લસ (+): પુરુષ (Male) માટે
🔴 ગોળ / માઈનસ (-): સ્ત્રી (Female) માટે
↔️ બે માથાવાળો તીર: પતિ-પત્નીના સંબંધ માટે
⬇️ ઊભી લીટી: માતા-પિતા અને સંતાનની પેઢી દર્શાવવા

૩. ઉંમર અને ઊંચાઈ સંબંધિત (Age & Height Problems)

ઉંમરના પ્રશ્નો માટે લોજિક
જો હાલની ઉંમર X હોય, તો t વર્ષ પહેલા: X - t
t વર્ષ પછીની ઉંમર: X + t

નોંધ: બે વ્યક્તિઓની ઉંમરનો તફાવત સમય સાથે ક્યારેય બદલાતો નથી.

૪. દિશા જ્ઞાન (Direction Sense Test)

દિશાઓનો નકશો અને અંતર

મુખ્ય ચાર દિશાઓ: ઉત્તર (North), દક્ષિણ (South), પૂર્વ (East), પશ્ચિમ (West).
પેટા દિશાઓ ઘડિયાળના કાંટાની દિશામાં: ઈશાન, અગ્નિ, નૈઋત્ય, વાયવ્ય.
ન્યૂનતમ અંતર માટે: પાયથાગોરસ પ્રમેય કર્ણ² = પાયો² + વેધ² નો ઉપયોગ કરવો.

૫. ઘડિયાળ અને કેલેન્ડર (Clock & Calendar)

ઘડિયાળ (Clock) શોર્ટકટ સૂત્રો
કલાક અને મિનિટ કાંટા વચ્ચેનો ખૂણો: θ = |(30×H) - (11/2×M)|
અરીસામાં પ્રતિબિંબ સમય (Mirror Image): ૧૧:૬૦ માંથી આપેલ સમય બાદ કરવો
કેલેન્ડર (Calendar) વધારાના દિવસો (Odd Days)

સામાન્ય વર્ષ = ૩૬૫ દિવસ = ૧ વધારાનો દિવસ
લીપ વર્ષ = ૩૬૬ દિવસ = ૨ વધારાના દિવસો
૧૦૦ વર્ષમાં વધારાના દિવસ = ૫ | ૨૦૦ વર્ષમાં = ૩ | ૪૦૦ વર્ષમાં = ૦

૭. ક્રમ અને સ્થાન (Ranking & Ordering)

સ્થાન નિર્ધારણ શોર્ટકટ
હરોળમાં કુલ વ્યક્તિઓની સંખ્યા: કુલ = ડાબેથી સ્થાન + જમણેથી સ્થાન - ૧
કોઈ એક બાજુથી સ્થાન શોધવું: સ્થાન = (કુલ - બીજી બાજુનું સ્થાન) + ૧

૮. સમાંતર શ્રેણી (Arithmetic Progression)

શ્રેણીના સૂત્રો
n મુ પદ શોધવા (Tₙ): a + (n - 1)d
n પદોનો સરવાળો (Sₙ): (n / 2) × [2a + (n - 1)d]

(a = પ્રથમ પદ, d = સામાન્ય તફાવત, n = પદોની સંખ્યા)

૧૩. સંભાવના (Probability)

સંભાવના સિદ્ધાંત
ઘટના A ની સંભાવના P(A) = શક્ય પરિણામો / કુલ પરિણામો
સિક્કા માટે કુલ પરિણામો (n સિક્કા): 2ⁿ
પાસા માટે કુલ પરિણામો (n પાસા): 6ⁿ
યાદ રાખો: સંભાવનાની કિંમત હંમેશા ૦ અને ૧ ની વચ્ચે જ હોય (0 ≤ P(A) ≤ 1).

બાકીના તાર્કિક પ્રકરણો અને વિશ્લેષણ (Chapter 9 to 20)

માહિતીનું અર્થઘટન (Data Interpretation - DI)

પાઈ-ચાર્ટ (Pie Chart) ગણતરી માટે મહત્વનો સંબંધ:
૧૦૦% = ૩૬૦° અને ૧% = ૩.૬°
DI ના પ્રશ્નો ઉકેલવા માટે ટકાવારી, સરેરાશ અને ગુણોત્તરના પ્રકરણ મજબૂત હોવા જરૂરી છે.

વિધાન અને અનુમાન / ગાણિતિક મોડેલીંગ

૧. તાર્કિક વિશ્લેષણ: આપેલા વિધાનને હંમેશા ૧૦૦% સત્ય માનીને જ તારણ ચેક કરવું.
૨. ગાણિતિક પ્રક્રિયા (Mathematical Operations): આપેલા ચિહ્નો (+ ને બદલે -, × ને બદલે ÷) બદલ્યા પછી BODMAS નિયમનો કડક અમલ કરવો.
